D3 Explanations

  • Select
    • Select Commands work by finding the first matching result in your file and using it for whatever comes next.
  • Append
    • The Append Command is a way to take whatever your next code is and add it to the end of the object that was opened in the pevious command.
  • Text
    • What ever is typed in this command will be the string that is used in the previous command.
  • Style Commands
    • The end which is a semicolon signifies the end of our code piece.
When all these elements are put together it creates something called a chain command which ,as can be seen above, works by starting a code which hands data over to your next command which does the same until you reach your end. For example d3.select(body).append(p).text(new paragraph);
